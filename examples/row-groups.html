Colspans in TBODY

If you support IE and you have a table where the first visible row contains cells with colspans such as the example below, you will need to pass an additional configuration option to floatThead. getSizingRow: function($table){...}.

getSizingRow should return a wrapped set of table cells within a table row. The length of the wrapped set should equal to the number of columns in the table.
By default IE8 uses the first visible row for sizing, but in this example we cannot do that because the first visible row contains a single TD.



{% highlight js %} $(function(){ $('#demo').floatThead({ scrollingTop: pageTop, getSizingRow: function($table){ // this is only called when using IE return $table.find('tbody tr:not(.grouping):visible:first>td'); } }); }); {% endhighlight %}
