Working with datatables

jquery.floatThead works out of the box with datatables or any other table framework.
Be sure to init the framework first, and then the plugin.

Code

{% highlight js %} $(document).ready(function () { var $table = $('#demo0'); $table.DataTable(); $table.floatThead({ scrollingTop: pageTop //i need this because of my floating header }); }); {% endhighlight %}

Using Datatable's FixedHeader plugin

Code

{% highlight js %} $(document).ready(function () { var table = $('#demo2').DataTable(); new $.fn.dataTable.FixedHeader(table); }); {% endhighlight %} This example shows the FixedHeader plugin written by datatable's author. I put it here to compare it with my plugin. FixedHeader does not support inner scrolling.
Also I couldn't figure out how set an offset of 41px from the top. Maybe it doesnt support this, or maybe I cant read docs.
